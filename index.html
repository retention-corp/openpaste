Two paths. One cleanup flow.

HTML path

Perplexity, Notion, Google Docs

Web apps put styled HTML on the clipboard. OpenPaste strips bulky inline styles, removes zero-width junk, and keeps links, headings, lists, and tables.

Text path

Claude Code, Warp, any terminal

Wrapped output gets stitched back into readable paragraphs while bullet lists, numbered steps, and fenced code blocks stay intact.
